# DESIGN.md

## 气质与意象

线条小狗（Line Puppy）——用最简练的黑色线条勾勒出憨态可掬的小狗形象，带着一丝笨拙和无限的温柔。这款恋爱记录应用就像一本手绘的恋爱手账：奶油色的纸页上，两只线条小狗并肩坐在公园长椅上看日落，身边散落着拍立得照片、手写的便签和标注了小心心的地图。

**关键词**：温暖、手绘感、柔软、治愈、日常甜蜜

## 视觉策略

- **图形语言**：大量使用手绘线条风格的装饰元素（小狗简笔画、小爪子印、爱心、小星星），以 SVG 内联方式呈现，保持轻盈感
- **摄影/图像方向**：照片展示区使用拍立得相纸风格——白色边框 + 轻微阴影 + 手写体日期标注
- **卡片设计**：圆角柔和（16-20px），微阴影带暖色调，像贴在手账本上的便签纸

## 配色方案

| 用途 | 色值 | 意象来源 |
|------|------|---------|
| 主色（温暖奶油） | `#FFF8F0` | 手账本内页的奶油色纸张 |
| 强调色（小狗棕） | `#C4956A` | 线条小狗常用的暖棕色 |
| 辅助色（柔粉） | `#F2C9C9` | 恋爱气息的淡粉色腮红 |
| 文字色（深棕） | `#4A3728` | 手写墨水的深褐色 |
| 点缀色（蜂蜜黄） | `#F5D98E` | 小狗项圈上的小铃铛色 |
| 背景色（奶白） | `#FEFAF5` | 摊开的手账本底色 |

## 字体排版

- **标题/展示字体**：`ZCOOL KuaiLe`（Google Fonts，中文字体，圆润活泼），用于页面大标题和纪念日卡片
- **正文字体**：`Noto Serif SC`（衬线，温暖文艺感），用于记录内容正文
- **UI 字体**：`PingFang SC` / 系统默认，用于标签、按钮等界面元素
- **排版节奏**：标题宽松字距（letter-spacing: 0.05em），正文行高 1.8，营造手账阅读感

## 动效与交互

- **运动性格**：柔和、轻盈，像翻动纸质手账页
- **缓动曲线**：`cubic-bezier(0.4, 0, 0.2, 1)` — 平滑但不拖沓
- **悬停效果**：卡片轻微上浮（translateY: -2px）+ 阴影加深，像用手指轻触便签纸
- **页面过渡**：淡入淡出（opacity + 轻微上移），模拟翻页
- **滚动体验**：内容区块依次淡入，带 100ms 的错开延迟

## 布局与响应式

- 移动端优先设计，最大内容宽度 480px（模拟手账本的竖版比例）
- 桌面端居中展示，两侧留白或装饰性线条小狗图案
- 底部导航栏（移动端 Tab Bar）：首页、时间线、空间、我的

## 设计禁忌

- 不要使用高饱和度的霓虹色或荧光色
- 不要使用直角卡片或硬阴影（违背手账柔和感）
- 不要使用过于复杂的渐变背景
- 不要使用冷色调蓝色系（与温暖主题冲突）
- 不要使用等宽字体或科技感的字体
- 不要使用 Material Design 的水波纹效果（太机械）
